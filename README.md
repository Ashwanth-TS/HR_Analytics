
# HR Analytics Dashboard | Power BI

## Overview
This project is an interactive **HR Analytics Dashboard** built in **Power BI** to analyze employee data and identify key workforce trends such as attrition, employee demographics, job satisfaction, and department-level patterns.

The goal of this project is to transform raw HR data into meaningful business insights through **data cleaning**, **DAX calculations**, and **interactive dashboard design**.

This project demonstrates practical skills in:
- Power BI
- Data cleaning and transformation
- DAX calculations
- HR data analysis
- Dashboard storytelling

---

## Project Objective
The main objective of this dashboard is to help understand:
- overall workforce composition
- employee attrition trends
- attrition by department, education field, gender, and age group
- employee job satisfaction patterns
- workforce demographics and employee distribution

---

## Dataset
The dataset contains employee-related HR information such as:
- Employee count
- Age
- Gender
- Department
- Education
- Education field
- Attrition
- Job role
- Job satisfaction
- Other workforce-related attributes

The dataset was loaded directly into Power BI for further cleaning and analysis.

---

## Tools Used
- **Power BI** – dashboard creation and visualization
- **Power Query** – data cleaning and transformation
- **DAX** – calculated columns, measures, and KPIs

---

## Data Preparation
After loading the dataset into Power BI, the data was cleaned and prepared for analysis.

### Data cleaning steps performed:
- Checked and corrected data types
- Removed unnecessary or inconsistent fields
- Handled missing or invalid values
- Standardized column names
- Created age bands and other grouped categories
- Prepared the dataset for KPI and visual analysis

---

## DAX Measures and Calculations
DAX was used to create important business measures and calculated fields for dashboard reporting.

### Examples of DAX used:
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Age Band classification
- Department-wise and gender-wise calculations

These measures helped in building KPI cards and detailed visual analysis.

---

## Dashboard Features
The dashboard provides a clear and interactive summary of HR performance using multiple visuals.

### Main dashboard highlights:
- **Overall Employees**
- **Attrition Count**
- **Attrition Rate**
- **Active Employees**
- **Average Age**

### Visual insights included:
- Department-wise attrition
- Number of employees by age group
- Job satisfaction rating by job role
- Education field-wise attrition
- Attrition rate by gender across different age groups
- Education-based filtering using slicers

---

## Key Insights
This dashboard helps identify important HR trends such as:
- departments with the highest attrition
- age groups where attrition is more common
- variation in attrition by gender
- education field contribution to attrition
- job roles with different satisfaction levels

These insights can support HR teams in improving retention, employee engagement, and workforce planning.
